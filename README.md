**ID**\
Sebuah cetak biru (blueprint) design system portabel berbasis Markdown yang memberikan batasan visual konkret kepada AI Coding Agents. Berfungsi sebagai lapisan desain instan agar AI dapat menghasilkan komponen UI yang konsisten, estetis, dan aksesibel secara modular tanpa prompt yang berulang.

**EN**\
A blueprint for a portable Markdown-based design system that provides concrete visual guidelines for AI coding agents. It serves as an instant design layer, enabling AI to generate consistent, aesthetically pleasing, and accessible UI components in a modular way without the need for repetitive prompts.
